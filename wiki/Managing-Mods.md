# Managing Mods

Learn how to browse, search, install, and manage your Farming Simulator mods.

## üîç Browsing Mods

### Main Mod Grid

When you open the app, you'll see a grid of available mods from multiple sources. Each mod card shows:

- **Thumbnail image**
- **Mod name**
- **Author**
- **Category**
- **Install button**

### Filtering by Category

Use the category sidebar to filter mods:

| Category | What It Includes |
|----------|------------------|
| All | All available mods |
| Tractors | Tractors and crawlers |
| Harvesters | Combines, forage harvesters |
| Trucks | Trucks, trailers, semi-trucks |
| Trailers | All types of trailers |
| Implements | Plows, cultivators, seeders |
| Maps | Custom maps and terrains |
| Objects | Placeable objects, decorations |
| Other | Miscellaneous mods |

### Searching

Use the search bar to find specific mods:

- Search by **mod name**: `John Deere 8R`
- Search by **author**: `Giants Software`
- Search by **keywords**: `trailer pack`

> **üí° Tip:** Search results come from all configured mod sources for your selected game.

## üì¶ Installing Mods

### Quick Install

1. Find a mod you want
2. Click the **Install** button
3. Wait for the download progress to complete
4. The mod is automatically placed in your game's mods folder

### What Happens During Installation

1. **Download** - The mod file is downloaded to a temporary folder
2. **Extract** - ZIP files are extracted (if applicable)
3. **Copy** - Mod files are moved to your game's mods folder
4. **Cleanup** - Temporary files are deleted

### Verifying Installation

After installing, check the **Installed Mods** section in the sidebar to confirm the mod appears.

## üîÑ Updating Mods

When a newer version of an installed mod is available:

1. The mod card will show an **Update** indicator
2. Click **Update** to download and install the latest version
3. The old version is replaced automatically

> **Note:** Update detection depends on the mod source providing version information.

## üí° Tips & Best Practices

### Mod Compatibility

- **Always check game version compatibility** before installing
- FS25 mods won't work in FS22, and vice versa
- Some mods require other mods as dependencies (check mod descriptions)

### Mod Folder Management

- Keep your mods folder organized
- Remove mods you no longer use to reduce load times
- Back up your mods folder before major changes

### Performance Considerations

- Too many mods can slow down your game
- Large maps and high-poly vehicle mods use more resources
- If experiencing issues, try disabling mods one by one
