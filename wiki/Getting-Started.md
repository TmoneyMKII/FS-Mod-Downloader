# Getting Started

This guide will walk you through setting up FS Mod Downloader for the first time.

---

## üì• Download & Installation

### Step 1: Download the Application

1. Go to the [Releases page](https://github.com/TmoneyMKII/FS-Mod-Downloader/releases/latest)
2. Download one of the packages:

| Package | Size | Best For |
|---------|------|----------|
| `FSModDownloader-vX.X.X-win-x64-standalone.zip` | ~150MB | **Recommended** - Works out of the box |
| `FSModDownloader-vX.X.X-win-x64.zip` | ~5MB | If you already have .NET 8 installed |

### Step 2: Extract the Files

1. Right-click the downloaded `.zip` file
2. Select **Extract All...**
3. Choose a destination folder (e.g., `C:\Games\FSModDownloader`)
4. Click **Extract**

### Step 3: Run the Application

1. Open the extracted folder
2. Double-click `FSModDownloader.exe`
3. If Windows SmartScreen appears, click **More info** ‚Üí **Run anyway**

## üéÆ Configure Your Games

### Automatic Detection

The easiest way to set up your games:

1. Click the **‚öôÔ∏è Settings** button (top-right corner)
2. Click **üîç Scan for Games**
3. The app will automatically find your Farming Simulator installations
4. Click **Save Settings**

The app checks these locations:
- `Documents\My Games\FarmingSimulatorXXXX\mods`
- Steam library folders
- GIANTS Software registry entries

### Manual Configuration

If auto-detection doesn't find your game:

1. Open **‚öôÔ∏è Settings**
2. Click **‚ûï Add Manual Entry**
3. Select your game version from the dropdown (FS15, FS17, FS19, FS22, FS25)
4. Click **Browse** and navigate to your mods folder
5. Click **Add**

> **üí° Tip:** The mods folder is usually located at:
> - `C:\Users\[YourName]\Documents\My Games\FarmingSimulator2025\mods`
> - Or within your Steam game installation folder

## üöÄ Your First Mod Installation

1. **Select your game** from the dropdown in the sidebar
2. **Browse or search** for a mod you want
3. Click the **Install** button on the mod card
4. Wait for the download to complete
5. **Launch your game** and enjoy the new mod!
